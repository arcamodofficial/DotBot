# Target Proyek dan Struktur ARCAMOD

1. Minimize Comments – Remove excessive or redundant comments.

2. Do Not Delete Intentionally Disabled Code – Keep any code that has been purposefully disabled.

3. Follow arcamod.cpp Style – Use static for internal functions.

4. Architectural Code – Maintain clean and professional code structure.

5. Preserve Functionality – Do not alter the behavior of any existing function.
