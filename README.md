## 📚 What is DotBot?

DotBot is a lightweight .NET AI assistant designed to enhance your daily activities. It connects to various editors through ACP (Agent Communication Protocol) and extends your tools using MCP (Modular Command Protocol). With DotBot, you can integrate skills and commands into your editor, command line interface (CLI), or chatbots. This makes managing your tasks easier and more efficient.
